Kresba vektorů

Vektorové kreslení, jak má být

Přejděte od skici k obrysu během okamžiku. Rychle zachyťte svůj nápad a snadno ho přeměňte v dokonalé vektory a dělejte vše, co vám Illustrator nikdy neumožnil: posouvejte, upravujte více řídicích bodů současně, zamykejte úhly, hromadně ovládejte křivost, harmonizujte segmenty křivek. Převezměte plnou kontrolu nad svými křivkami.

Přečtěte si tutoriály
  • Sketching
  • Drawing good paths

Interpolace

Pracujte méně, tvořte více: interpolujte své kresby

Kreslete a spravujte kresby více masterů a poté snadno vytvořte tolik meziřezů, kolik potřebujete.

Přečtěte si tutoriály
  • Multiple Masters, part 1: setting up masters
  • Multiple Masters, part 2: keeping your outlines compatible
  • Multiple Masters, part 3: setting up instances

Přechod od tenkého k tučnému není vždy lineární: vložte mezimastery pro větší přesnost interpolace. Někdy se tvary musí změnit, když se stávají tučnějšími. Přepínejte mezi paralelními interpolacemi pomocí alternativních masterů.

Přečtěte si tutoriál
  • Switching shapes

Převeďte svůj multi-masterový projekt na variabilní font jediným kliknutím. Ovládejte podmnožiny a OT funkce pomocí nastavení exportu variabilního fontu.

Přečtěte si tutoriál
  • Creating a variable font

Opakované tvary

Přestaňte ztrácet čas kopírováním a vkládáním, využívejte své tvary chytře

Využívejte glyfů jako komponent uvnitř dalších glyfů: upravte originál a všechny propojené komponenty se okamžitě přizpůsobí. Využijte to, aby vaše a bylo stejné v äâãāăȧà, nebo pro synchronizaci libovolného počtu tvarů napříč mnoha glyfů.

Přečtěte si tutoriály
  • Diacritics
  • Reusing shapes: component tricks
  • Advanced diacritics: narrow marks
  • Advanced diacritics: adapted base letters
  • Creating a pixel font

Aplikujte rohové komponenty na rohové uzly — skvělý způsob, jak ovládat, dolaďovat a synchronizovat zaoblení rohů, výběžky a patky. Vytvářejte transformovatelné tvary s chytrými komponentami: představte si komponentu, kterou často potřebujete, ale která se musí v každé situaci trochu přizpůsobit.

Přečtěte si tutoriály
  • Reusing shapes: corner components
  • Reusing shapes: smart components

Rozšiřte vektorové segmenty na libovolný tvar pomocí štětcových komponent: ideální pro modulární písma nebo grafické prvky jako šipky. Nebo znovu využijte stejné hrany tvarů pomocí segmentových komponent.

Přečtěte si tutoriály
  • Reusing shapes: brushes
  • Reusing shapes: segment components

Barvy a kontury

Kdo řekl, že fonty jsou jen černobílé?

Aplikujte barvy, stíny, záře a přechody s novými barevnými vrstvami. Ideální pro informační grafiku a barevná písma. Nebo využijte možností formátů barevných fontů: SVG, CPAL/COLR, sbix a klasické vrstvové fonty.

Přečtěte si tutoriály
  • Creating an SVG color font
  • Creating a Microsoft color font (CPAL/COLR)
  • Creating a layered color font
  • Creating an Apple color font (sbix)

Přidejte tahy ke svým cestám a ty se nedestruktivně rozšíří na vyplněné obrysy. Dynamicky odečítejte cesty od jiných tvarů tím, že je přeměníte na masky. Vytvářejte pixelově přesné kontury aplikováním tahů specifických pro danou velikost: udělejte tu čáru dva pixely širokou, bez ohledu na to, v jakém měřítku ikonu exportujete.

Přečtěte si tutoriál
  • How to make a monoline font

Správa

Žonglujte s tisíci glyfů a mějte vše pod kontrolou

Řaďte, filtrujte a prohlížejte v Zobrazení fontu: najděte jakýkoli glyf, který potřebujete, během okamžiku.

Přečtěte si tutoriály
  • Adding glyphs to your font
  • Batch renaming

Hromadně upravujte mnoho glyfů najednou: měňte informace o glyfech a boční přístupy, aplikujte filtry a transformujte libovolný počet písmen, číslic a symbolů.

Vytvářejte a spravujte varianty kompletních projektů v samostatném souboru bez duplikování dat: podmnožiny webfontů, kompletní sady fontů s jiným názvem rodiny písma? Vy rozhodujete.

Export

Exportujte jakýkoli hlavní formát fontů nebo obrázků

Uspokojte všechny potřeby fontů dodáním všech formátů OpenType: OTF s CFF, TTF, stejně jako webfonty WOFF a WOFF2. Vyměňujte data nebo se začleňte do stávajícího pracovního postupu exportem UFO a Font Metrics.

Přečtěte si tutoriály
  • Working with UFO
  • Webfonts

Vylepšete svůj projekt v Xcode pomocí dokonalých SFSymbolů. Vytvářejte sady ikon, dingbatů a symbolů. Exportujte je jako ikonové fonty, nebo jako PNG, SVG nebo PDF.

Přečtěte si tutoriál
